# Homebrew Tap for Kin Code

This is the official Homebrew tap for [Kin Code](https://github.com/kinra-ai/kin-code).

## Installation

```bash
brew tap kinra-ai/tap
brew install kin-code
```

## Updating

```bash
brew update
brew upgrade kin-code
```

## Troubleshooting

If you encounter issues, try:

```bash
brew untap kinra-ai/tap
brew tap kinra-ai/tap
brew install kin-code
```
